# Heckler Tooling

A docker container captures all dependencies to build heckler kernels, vmm and tools.

### Build Container
```
./docker-build.sh
```
Alternatively, a prebuilt container is pushed to [docker hub](https://hub.docker.com/r/beancd/ahoi-heckler/tags).

### Enter Container

```
./docker-run.sh
```

### Remove Container
```
./docker-rm.sh
```
